# 🌐 3-in-1 Utility Web App

Built for my IT Ausbildung portfolio. I made this using strictly **pure HTML, CSS, and Vanilla JavaScript**. No heavy frameworks used.

👉 **Live Demo:** [https://notbeatle.github.io/Beatlecalc/](https://notbeatle.github.io/Beatlecalc/)

### Key Features
*   **Calculator:** Standard math with a running history feature.
*   **BMI Calculator:** Color-coded health results based on height/weight.
*   **Live Currency:** Real-time rates using ExchangeRate API (`async/await`).
*   **Dark Mode:** Saves your preference to `localStorage`.

### 📱💻 Fully Responsive & Browser-Based
This is a lightweight web app that runs directly in your browser. I have optimized it so it looks and works perfectly on **any screen size**—whether you are using a mobile phone, a tablet, or a desktop PC. 

### 📦 Android App Version
I also packaged the web app into a native Android app!
1. Download the `.apk` from this repo.
2. Allow "Install from unknown sources" in settings.
3. Install and run from your home screen.

### How to Run Locally
Download the ZIP -> Extract -> Open `index.html` in any browser. *(Needs internet for live currency rates).*
