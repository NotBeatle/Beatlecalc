# Security Policy

## Supported Versions

Currently, only the latest release of Beatle Calc is supported with security updates. Since this app operates 100% offline, the attack surface is minimal, but maintaining code integrity remains a priority.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security and privacy are core features of this application. If you happen to discover a security vulnerability within Beatle Calc, please **do not open a public issue.**

Instead, please report it privately to ensure it can be patched before becoming public knowledge. You can reach out directly via:

* **Email:** Prathameshsathe3101@gmail.com
* **Instagram Direct Message:** [@Notbeatle](https://instagram.com/Notbeatle)

Please include the following in your report:
* A description of the vulnerability.
* Steps to reproduce the issue.
* The device and Android version (or browser) you were using.

I will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a timeline for the fix.
